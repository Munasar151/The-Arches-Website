ITQ

If you have some basic computer experience, this course will help you develop a range of skills and knowledge relevant to the workplace and gain a Level 1 or Level 2 qualification.

There are a range of units, each covering a different software application or task. We will be using the latest version of Windows 10 and Microsoft Office 2016:

  • Word Processing
  • Spreadsheets
  • Database
  • Presentations (Powerpoint)
  • Internet and Email
